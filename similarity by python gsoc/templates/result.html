About the Similarity Metrics

Histogram Similarity: Compares the color distribution between images. Higher scores indicate similar color profiles.

Structural Similarity (SSIM): Measures the perceived similarity in structure, luminance, and contrast. A score of 100% means identical images.

Feature Matching: Uses SIFT algorithm to detect and match distinctive features between images. Higher scores indicate more matching features.
